The "internal variables" for the "ReminderNotes.js" file.

Coding guidelines are important.

To know the difference of an internal variable and function from an external variable and function we start every internal variable and function with a underscore.

You are building a custom widget based on top of a DOJO widget.

Variabels from DOJO can be set.

Properties that are set in the ReminderNotes.xml are avaliable on the "this" scope.

Important to know is that we are going to import a template with templateString.
