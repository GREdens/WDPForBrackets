The "_renderReminderNotesResult" internal function.

We use a template to render a note.

Aggregation data with Mendix will be done asynchronised and by calling microflows in a loop.

Data aggregated is always unsorted.

Async function calls need to have passed the current scope through 'lang.hitch'.

A microflow can only give back a list of one type of objects.

Objects over assiociation are not avaliable and need to be aggregated.
